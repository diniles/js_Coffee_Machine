Array Creation

Create a function named createNewArray and it accepts three arguments, i.e. value, start, end. Use the originalArr and fill() method and then return the newly created array.


Sample Input:
3
0
8

Sample Output:
[ 3, 3, 3, 3, 3, 3, 3, 3 ]



Memory limit: 256 MB
Time limit: 15 seconds
